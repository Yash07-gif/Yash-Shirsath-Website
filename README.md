# 🌐 Yash Shirsath – Personal Portfolio Website

Welcome to the official GitHub repository of my **Personal Portfolio Website**.
This project represents my **professional identity**, showcasing my skills, projects, experience, and passion for **Data Analytics, Artificial Intelligence, and Software Development**.

## 📌 Overview

In today’s competitive tech landscape, a strong digital presence is essential.
This portfolio website is designed to serve as a **central hub** where recruiters, hiring managers, and collaborators can explore:

* My professional background
* Technical and analytical projects
* Tools & technologies I work with
* Articles and learnings
* Resume and contact details

The website is **fully responsive**, visually clean, and easy to navigate across desktop, tablet, and mobile devices.

## 🎯 Purpose of the Project

The main goals of this project are:

* To present my **skills and projects** in a structured manner

* To provide easy access to my **resume and contact information**

---

## 🛠️ Technologies Used

This portfolio is built using core web technologies:

### Frontend

* **HTML5** – Website structure and semantic layout
* **CSS3 / SCSS** – Styling, responsiveness, and animations
* **JavaScript** – Interactivity and dynamic behavior

### Tools & Utilities

* **SCSS Preprocessor** – Cleaner and modular styling
* **Prepros** – SCSS compilation and asset management
* **Vercel** – Deployment and hosting
